# Hairfolio 레퍼런스 분석 및 구현 가이드

## 분석 대상
- `design.png` 브랜드 보드 (로고/타이포/컬러)
- 로고 단일 샘플 이미지 (`Hairfolio` 워드마크)
- `home` 레퍼런스 (히어로 + 오버레이 카드)
- `searchDesigner` 레퍼런스 (좌측 필터 + 창문형 카드 그리드)
- `login.png` (좌우 분할 로그인)
- `designerprofile.png` (대형 히어로 + 정보 스트립 + 포트폴리오 섹션)
- `custompage.png` (디자이너 페이지 커스텀 편집기)
- `salonpage.png` (살롱 상단 3영역)

## 브랜드 토큰
- 로고: 워드마크 `Hairfolio`를 모든 페이지 좌상단에 텍스트로 사용하고 배경 박스/흰 캔버스는 노출하지 않음
- 폰트: 한글 UI는 `Pretendard`, 영문 브랜드 강조는 `Poppins` 보조
- 기본 컬러: `#111111`, `#333333`, `#CDB79E`, `#EDE7DF`, `#FFFFFF`
- 보조 컬러: `#E7C1D1`, `#B8D2BE`, `#A7C7E0`, `#DCC8E6`, `#F5A5A5`
- 라운드: 대형 패널 28px, 카드 24px, 입력/버튼 16px, 칩/태그 pill
- 그림자: 부드러운 웜톤 그림자(과한 대비/깊이 금지)

## 페이지별 목표

### 0. `design.md` 작성
- 레퍼런스 기반 토큰/구성 원칙 명시
- 페이지별 구현 우선순위와 수용 기준 명시

### 1. 공통 로고
- 모든 페이지 좌상단 로고는 텍스트 워드마크로 통일
- 배경과 구분되는 흰 사각형 이미지 자산 사용 금지

### 2. 홈 히어로
- 배경 이미지 위에 콘텐츠가 얹히는 구조
- `헤어모델 모집 중`, `오늘의 인기 스타일` 카드는 glassmorphism 카드로 오버레이
- 검색 바는 히어로 내부에 결합

### 3. 디자이너 찾기
- 좌측 패널: 타이포/버튼 비율 확대, 필터 그룹 시각적 구분 강화
- 우측 카드: 창문형(상단 곡면) 이미지 프레임 적용
- 카드 메타정보(태그/좋아요/예약가능시간) 위계 정리

### 4. 로그인
- 좌우 분할 레이아웃(브랜딩 비주얼 + 로그인 폼)
- `일반 로그인 / 디자이너 로그인` 분기 제공
- 휴대폰/이메일 탭 + 소셜 로그인 영역 구성

### 5. 디자이너 상세
- 대형 히어로 + 정보 스트립 + 탭형 섹션 헤더 구성
- 포트폴리오 섹션 그룹화 및 CTA 반복 배치

### 6. 디자이너 커스텀
- 디자이너 로그인 시 편집 모드 진입
- 페이지 타이틀/헤드라인/소개/테마/버튼 문구 수정 가능
- 저장 후 일반 사용자용 디자이너 상세에 반영

### 7. 살롱 상세
- 상단을 `대표 이미지 / 살롱 정보 / 평점 패널` 3영역으로 구성
- 소속 디자이너, 대표 스타일 섹션을 레퍼런스 비율에 맞춰 재배치

### 8. 헤어모델 모집
- `지원하기` 버튼 제거
- 카드 클릭 시 디자이너 또는 디자이너 준비생 프로필로 이동

## 반응형 기준
- Desktop: 레퍼런스 비율 최대한 유지
- Tablet: 우측 보조 패널은 아래로 드롭 가능
- Mobile: 히어로/필터/카드 1열 우선, 핵심 CTA 가독성 유지
